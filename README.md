## 🧐 프로젝트 소개

**Monico**는 사용자가 암호화폐의 실시간 시세, 관련 뉴스, 그리고 커뮤니티 기능을 통해 다른 사용자들과 정보를 교류할 수 있는 웹 애플리케이션입니다. 깔끔하고 직관적인 UI를 통해 사용자에게 필요한 정보를 효과적으로 전달하는 것을 목표로 합니다.

## ✨ 주요 기능

- **🪙 실시간 시세 정보**: Upbit API를 활용하여 주요 암호화폐의 실시간 가격 및 등락률 정보를 제공합니다.
- **📈 차트 시각화**: 그래프를 사용하여 암호화폐의 상세한 캔들 차트를 제공합니다.
- **📰 최신 뉴스**: 암호화폐와 관련된 최신 뉴스를 모아 보여줍니다.
- **👥 커뮤니티**: 사용자들이 자유롭게 게시글을 작성하고 댓글을 통해 소통할 수 있는 공간을 제공합니다.
- **👤 사용자 프로필**: 자신의 정보 및 활동 내역(작성글 등)을 관리할 수 있습니다.
- **❤️ 관심 목록**: 원하는 암호화폐를 관심 목록에 추가하여 쉽게 확인할 수 있습니다.
